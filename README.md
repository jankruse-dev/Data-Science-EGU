### Infos von Segger 

- Captive Soil Moisture Sensor ADC Sensor für Feuchte und Temperatur Messung --> liefert 0 - 3 V Spannungssignal
- Rohwert im Vergleich zur Bodenfeuchtigkeit
- Kalibrieren der Feuchtigkeit entweder mit Abwiegen 
- I2C --> Internes Bus-System 
- Bibliotheken für ESP von dem Hersteller 
- BME 680 oder BME 280 
- [OpenWeathermap](https://openweathermap.org/) für Wetterdaten kann man über URL Abrufen. In Windows über Aufgabenplanung kann man das Skript triggern 
- [TTN](https://www.thethingsnetwork.org/) --> The Things Network, LoRaWan Antenne
- Pumpenkennlinie kann man anlernen --> dauert lange oder man macht einen linearen Ansatz. 
- Relais für Pumpe braucht man auch weil sonst der Strom zu groß wird --> Low Level Trigger Relais 5x20 mm Schmelzsicherung + Halterung, wäre auch im Labor zu bekommen
- V_in Pin abgesichert anschließen, wenn Programm aufgespielt wird nicht mit seperatem Netzteil anschließen 
- FH iot Server zum Speichern der Daten? --> Steffen fragen
